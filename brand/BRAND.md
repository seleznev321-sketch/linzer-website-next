## 1. Логотип — принцип построения

- Логотип — слово **LINZER** прописными буквами, разбитое на **4 строки** внутри сетки **4×4 (16 ячеек)**.
- Перенос и разрядка букв — случайны, НО:
  - геометрический центр каждой буквы совпадает с центром одной из 16 ячеек;
  - в каждой строке и каждом столбце — минимум одна буква.
- За счёт этого при любой раскладке считывается чёткий **квадрат** — это и есть константа.
- Существует **16 официальных вариантов** раскладки. Применение разных вариантов на одном носителе — приём узнаваемости (зритель запоминает «игру букв», а не одну форму).
- Шрифт логотипа — **Gotham**, прописные.
- Визуальный эталон 16 вариантов — `pages_reading/p05.png` (буквы чёрные, ячейки сетки тонко обведены красным/синим).
- Обложка-эталон — `pages_reading/p01.png`: чёрный фон, белая сетка `+`, буквы LIZER/LINZER вразброс.

### Готовый пример для остекления (прямой референс hero сайта)
`pages_reading/p11.png` — обложка буклета компании: **монохромное бирюзовое фото стеклянного дома** на весь блок, поверх крупный заголовок **«ВАШ СТЕКЛЯННЫЙ ДОМ»**, набранный по принципу сетки 4×4 (буквы вразброс), слоган *«живите с прекрасными видами»*, мелкий логотип LINZER-сеткой и `www.mylinzer.ru`. Это образец того, как стиль ложится на тему остекления — брать за основу первого экрана.

## 2. Фирменная текстура («миллиметровка»)

- Сетка с квадратными ячейками; узлы помечены маленькими **крестиками `+`**.
- Ассоциация — **миллиметровая бумага** → точность архитектурных/инженерных разработок.
- Это сигнатурный элемент: используется как фон (см. обложку буклета и стр. 6/8).
- На 3D-объектах узлы сетки распределяются по поверхности и подчёркивают форму.

## 3. Узор

- Сам логотип в крупном масштабе = абстрактная композиция из фирменных фигур (геометрия букв).
- Приём «через призму логотипа»: ч/б фото здания + крупные яркие буквы поверх.

## 4. Цвет — ключевое правило

> **Цвет = активный ФОН** для логотипа и элементов.
> В одном макете — **не более одного оттенка**.
> Второй активный цвет — **всегда глубокий чёрный или белый**.

Палитра (CMYK из буклета → приближённый HEX, наивная конвертация — **калибровать перед продакшеном**):

| Цвет | CMYK | ≈ HEX |
|------|------|-------|
| Жёлтый | C0 M10 Y100 K0 | `#FFE600` |
| Лайм | C40 M0 Y90 K0 | `#99FF1A` |
| Бирюзовый | C70 M0 Y40 K0 | `#4DFF99` |
| Оранжевый | C0 M50 Y90 K0 | `#FF801A` |
| Чёрный (rich) | C30 M30 Y30 K100 | `#000000` |
| Голубой | C90 M30 Y0 K0 | `#1AB3FF` |
| Красный | C0 M90 Y70 K0 | `#FF1A4D` |
| Пурпурный | C30 M90 Y0 K0 | `#B31AFF` |
| Индиго | C80 M80 Y0 K0 | `#3333FF` |

Логика системы: **одна сочная плашка-фон + чёрный/белый + логотип**. Каждый носитель «звучит» одним цветом.

## 5. Шрифты

| Назначение | Шрифт |
|------------|-------|
| Заголовочный | **GothamPro-Black** |

Вся акциденция и текст — одна гарнитура **Gotham**.
Веб-замена (Gotham не бесплатна): близкие гео-гротески — **Montserrat / Manrope / Mont / Gilroy**. Подобрать перед применением.

## 6. Формат и тон

- Любимый формат вёрстки — **«два квадрата», соотношение 1:2**.
- Стилистика: минимализм, точность «миллиметровки», hi-tech, монохромное фото + яркие буквы.
- Носители из буклета: буклет, билборд, вывеска, папка, визитка, бланк, шильд, чашка, карандаш, авто.
